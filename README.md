# GingSeng 人蔘
人蔘為一款快速卡牌養成遊戲：
* 4個主線
* 4種卡牌類型，包含日常生活、趣味性、無厘頭等等超過百種事件
* 高達90種結局，等你來收集

快加入人蔘，重活你的一生

## 遊戲說明
利用卡牌事件的選擇來影響人物數值，當任一數值歸零或超標即結束遊戲，並獲得一種結局
* 普通卡－普通事件
* 主線卡－跟最初選擇的志願有關的事件
* 餵食卡－與食物有關
* 鬼牌卡－強制觸發特殊結局
